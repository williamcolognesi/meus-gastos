
# App - Meus gastos


## Visão geral

- **O que é:** Mini sistema de controle financeiro pessoal.
- **Publíco alvo:** Pessoas que querem entender melhor os gastos.
- **Objetivo:** Clareza nos gastos e entender pra onde o dinheiro vai.



## Requisitos Funcionais
O que o sistema deve fazer:

- **RF01:** O usuário deve conseguir criar uma conta e fazer login.
- **RF02:** O usuário deve cadastrar transações (valor, descrição, data, categoria).
- **RF03:** O usuário deve conseguir editar ou excluir transações.
- **RF04:** O sistema deve exibir o saldo atual (Receitas - Despesas).
- **RF05:** O sistema deve permitir filtrar transações por mês/ano.
- **RF06:** O sistema deve exibir um resumo visual (gráfico) por categoria.
## Requisitos Não Funcionais
Como o sistema **deve ser**:

- **RNF01:** Interface responsiva (Mobile First, para cadastrar gastos na rua).
- **RNF02:** Performance (carregamento em menos de 2s).
- **RNF03:** Segurança (dados sensíveis protegidos no banco).
